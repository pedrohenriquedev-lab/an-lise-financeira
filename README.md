# Sistema de Análise Financeira Empresarial

Projeto acadêmico desenvolvido em equipe para a faculdade, com foco em Programação Orientada a Objetos (POO) e lógica pura em Python.

## Sobre o projeto

Este sistema permite cadastrar dados financeiros de uma empresa, calcular indicadores econômicos e gerar um diagnóstico com base em critérios simples de análise financeira.

O projeto foi desenvolvido com o objetivo de praticar:
- estruturação de classes
- organização de código
- cálculo de indicadores financeiros
- análise de desempenho empresarial
- exibição de resultados em formato de dashboard no terminal

## Funcionalidades

- Cadastro de dados financeiros por ano
- Cálculo de indicadores como:
  - liquidez corrente
  - liquidez seca
  - liquidez imediata
  - endividamento geral
  - ROE
  - margem líquida
- Diagnóstico financeiro com score
- Classificação de liquidez, endividamento e ROE
- Dashboard com histórico dos dados inseridos

## Tecnologias utilizadas

- Python
- Programação Orientada a Objetos
- Lógica de programação
- Terminal / console
